## Environment variables

Set secrets with `wrangler secret put <NAME>` in production. Non-sensitive values go in `wrangler.jsonc` under `[vars]`.

### Required

| Variable | Example | Description |
|---|---|---|
| `UPSTREAM_URL` | `https://api.openai.com/v1` | Upstream base URL. See [Gateway setup](/docs/gateways/openai) for examples per provider. |
| `OPENAI_API_KEY` | `sk-...` | Injected as `Authorization: Bearer` on OpenAI routes. Required for OpenAI routes. |

### Provider keys (set the ones you use)

| Variable | Routes | Description |
|---|---|---|
| `ANTHROPIC_API_KEY` | `/v1/messages` | Injected as `x-api-key` on Anthropic routes. |
| `AZURE_OPENAI_API_KEY` | `/azure/v1/*` | Injected as `api-key` header on Azure routes. Required to enable Azure. |
| `AZURE_OPENAI_ENDPOINT` | `/azure/v1/*` | Azure resource base URL, e.g. `https://my-resource.openai.azure.com`. Required to enable Azure. |
| `AZURE_OPENAI_API_VERSION` | `/azure/v1/*` | Azure API version. Defaults to `2024-02-01` when unset. |
| `GEMINI_API_KEY` | `/gemini/v1/*` | Injected as `Authorization: Bearer` on Gemini routes. Required to enable Gemini (unless using Vertex AI via `GEMINI_UPSTREAM_URL`). |
| `GEMINI_UPSTREAM_URL` | `/gemini/v1/*` | Override Gemini upstream URL. Default: `https://generativelanguage.googleapis.com/v1beta/openai`. Use for Vertex AI. |

### Gateway auth

| Variable | Example | Description |
|---|---|---|
| `CF_AIG_TOKEN` | `vck_...` | Cloudflare AI Gateway auth token. Accepts bare token or `Bearer token` â€” normalized automatically. Leave empty if gateway auth is disabled. |

### Compression tuning

| Variable | Default | Description |
|---|---|---|
| `TOON_THRESHOLD` | `0.6` | Minimum eligibility score (0â€“1) before encoding. Lower = more aggressive. |
| `TOON_THRESHOLD_CHAT` | _(inherits `TOON_THRESHOLD`)_ | Per-route override for chat completions across all providers (`/v1/chat/completions`, `/azure/v1/chat/completions`, `/gemini/v1/chat/completions`). |
| `TOON_THRESHOLD_EMBEDDINGS` | _(inherits `TOON_THRESHOLD`)_ | Per-route override for embeddings across all providers (`/v1/embeddings`, `/azure/v1/embeddings`, `/gemini/v1/embeddings`). |
| `TOON_THRESHOLD_MESSAGES` | _(inherits `TOON_THRESHOLD`)_ | Per-route override for Anthropic `/v1/messages`. |
| `TOON_DRY_RUN` | `false` | When `true`, estimates savings but sends original payload. Great for testing before enabling compression on production traffic. |
| `TOON_LOG_SAVINGS` | `true` | Write per-request savings rows to D1. Set `false` to skip D1 writes entirely. |

### Reliability

| Variable | Default | Description |
|---|---|---|
| `UPSTREAM_TIMEOUT_MS` | `30000` | Abort upstream fetch after this many milliseconds. Returns `504` to client on timeout. |

### Observability & admin

| Variable | Description |
|---|---|
| `ADMIN_KEY` | Protects all `/savings/*` routes. If not set, those routes return `404`. Set to a random 32+ char string. |
| `PROXY_AUTH_KEY` | **Required for production.** When set, all proxy routes (`/v1/*`, `/azure/v1/*`, `/gemini/v1/*`) require `Authorization: Bearer <value>`. When unset, all proxy routes return `401`. Set to a random 32+ char string. |

## Updating model pricing

The USD savings estimate in the savings dashboard uses a static pricing table bundled at deploy time. To refresh it with the latest prices from [LiteLLM's model registry](https://github.com/BerriAI/litellm/blob/main/model_prices_and_context_window.json):

```bash
npm run generate:pricing  # fetches latest prices â†’ src/lib/pricing.ts
npm run deploy
```

Run this whenever new models are added or prices change. The table covers 245+ models across OpenAI, Anthropic, Gemini, Mistral, Groq, DeepSeek, and Cohere.

## Threshold tuning guide

Not sure what threshold to use? Start with dry-run mode:

<Steps>
  <Step title="Enable dry-run mode">
    Set `TOON_DRY_RUN=true` in `.dev.vars` or `wrangler.jsonc`.
  </Step>
  <Step title="Run your real traffic">
    Use toongate normally for a day or two. No payloads are compressed â€” but savings are estimated and logged.
  </Step>
  <Step title="Query your savings log">
    ```bash
    npx wrangler d1 execute toongate-savings --local \
      --command "SELECT AVG(tokens_saved * 1.0 / tokens_before) as avg_ratio,
                        AVG(tokens_saved) as avg_saved
                 FROM savings"
    ```
  </Step>
  <Step title="Choose your threshold">
    If `avg_ratio` is high (>0.3) across most requests â†’ lower threshold to 0.4â€“0.5. If mixed â†’ keep at 0.6. If low â†’ your traffic is mostly free-form text; compression won't help much.
  </Step>
  <Step title="Disable dry-run">
    Remove `TOON_DRY_RUN` or set to `false`. Real compression starts immediately.
  </Step>
</Steps>

## Example: `.dev.vars` (full)

```bash
# OpenAI / OpenAI-compatible  (/v1/*)
UPSTREAM_URL=https://api.openai.com/v1
OPENAI_API_KEY=sk-proj-...

# Anthropic  (/v1/messages)
ANTHROPIC_API_KEY=sk-ant-...

# Azure OpenAI  (/azure/v1/*)
AZURE_OPENAI_API_KEY=...
AZURE_OPENAI_ENDPOINT=https://my-resource.openai.azure.com
# AZURE_OPENAI_API_VERSION=2024-02-01  # optional, this is the default

# Gemini  (/gemini/v1/*)
GEMINI_API_KEY=AIza...
# GEMINI_UPSTREAM_URL=  # optional, for Vertex AI

# Compression
TOON_THRESHOLD=0.6
TOON_DRY_RUN=false
TOON_LOG_SAVINGS=true

# Proxy auth â€” required for production, protects all proxy routes
PROXY_AUTH_KEY=change-me-to-a-random-32-char-string

# Admin â€” protects /savings/* routes
ADMIN_KEY=change-me-to-a-random-32-char-string

# Reliability
UPSTREAM_TIMEOUT_MS=30000
```
