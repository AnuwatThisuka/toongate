How it works

Your App │ OpenAI SDK (unchanged) ▼ toongate ← compresses JSON arrays to TOON (~40% fewer tokens) │ forwards all gateway headers untouched ▼ Any Gateway ← CF AI Gateway / Helicone / LiteLLM / direct — your choice │ ▼ OpenAI / Anthropic ← receives smaller payload; bills you less │ ▼ (response) toongate ← passes response through, logs savings to D1 │ ▼ Your App ← sees normal JSON, no TOON awareness needed

toongate encodes only when it helps. Uniform arrays of objects (RAG chunks, DB rows, product lists) compress well. Free-form text and deeply nested configs pass through unchanged.
