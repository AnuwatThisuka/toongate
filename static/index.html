How it works
Four steps.
Invisible to your app.

toongate runs at the edge, compresses, forwards, and passes the response back. Your app sees standard JSON — no TOON awareness needed.

01
Receive request
Your SDK sends a normal OpenAI/Anthropic request to toongate instead of directly to the provider.
02
Score & compress
Eligibility scorer evaluates the payload (0–1). Uniform arrays above the threshold are encoded to TOON.
03
Forward upstream
Compressed request goes to your gateway or provider. All existing headers pass through untouched.
04
Log & return
Response returned to your app unchanged. Savings written to D1 fire-and-forget — never blocks the response.

Reliability
Production-grade
by default.

toongate never adds new failure modes to your pipeline. Every feature has a safe fallback — compression errors, upstream failures, and DB writes are all handled gracefully.

01
Circuit breaker
Trips automatically at 50% error rate (last 20 requests). Falls back to raw pass-through and recovers after 30 s.
02
Auto-retry
Retries once on 429 / 503, respecting the upstream retry-after header. Capped at 10 s.
03
Adaptive threshold
Set TOON_THRESHOLD_AUTO=true to auto-tune compression aggressiveness from your real savings history.
SDKs
3 SDKs.
Same one-line change.

No new methods to learn. Just swap the baseURL — everything else stays the same. 

import OpenAI from "openai"

const openai = new OpenAI({
  baseURL: "https://toongate.workers.dev/v1",
  // apiKey injected by toongate from OPENAI_API_KEY env
})

// Use exactly as before — streaming, tools, everything works
const res = await openai.chat.completions.create({
  model: "gpt-4o",
  messages: [{ role: "user", content: myRagChunks }],
})

// Check savings in response headers:
// x-toongate-tokens-saved: 569
// x-toongate-compressed: true
from openai import OpenAI

client = OpenAI(
    base_url="https://toongate.workers.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": rag_chunks}]
)

# Savings logged to D1 automatically
import "github.com/sashabaranov/go-openai"

config := openai.DefaultConfig(os.Getenv("OPENAI_API_KEY"))
config.BaseURL = "https://toongate.workers.dev/v1"

client := openai.NewClientWithConfig(config)

resp, _ := client.CreateChatCompletion(ctx,
    openai.ChatCompletionRequest{
        Model:    openai.GPT4o,
        Messages: messages,
    },
)
