---
title: "POST /azure/v1/embeddings"
api: "POST https://toongate.workers.dev/azure/v1/embeddings"
description: Azure OpenAI embeddings with automatic TOON compression.
---

## Overview

Proxies to `{AZURE_OPENAI_ENDPOINT}/openai/deployments/{model}/embeddings?api-version={AZURE_OPENAI_API_VERSION}`. The deployment name is taken from the `model` field.

## Authorization

<ParamField header="Authorization" type="string">
  Required only when `PROXY_AUTH_KEY` is set. Pass `Bearer <PROXY_AUTH_KEY>`. The Azure API key is injected automatically.
</ParamField>

## Request body

<ParamField body="model" type="string" required>
  Azure **deployment name** for your embedding model (e.g. `my-embedding-deployment`).
</ParamField>

<ParamField body="input" type="string | array" required>
  Text or array of texts to embed. Arrays of structured objects are eligible for TOON compression.
</ParamField>

## Response headers

| Header | Description |
|---|---|
| `X-Toongate-Compressed` | `true` if the request payload was compressed |
| `X-Toongate-Tokens-Before` | Estimated token count before compression |
| `X-Toongate-Tokens-After` | Estimated token count after compression |
| `X-Toongate-Tokens-Saved` | Tokens saved |

<RequestExample>
```bash cURL
curl https://toongate.workers.dev/azure/v1/embeddings \
  -H "Content-Type: application/json" \
  -d '{
    "model": "my-embedding-deployment",
    "input": "The quick brown fox"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "object": "list",
  "data": [{ "object": "embedding", "embedding": [0.01, -0.02, ...], "index": 0 }],
  "model": "text-embedding-ada-002",
  "usage": { "prompt_tokens": 5, "total_tokens": 5 }
}
```

```json 503
{ "error": "Azure OpenAI is not configured" }
```
</ResponseExample>
