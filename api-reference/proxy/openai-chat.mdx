---
title: "POST /v1/chat/completions"
api: "POST https://toongate.workers.dev/v1/chat/completions"
description: OpenAI-compatible chat completions with automatic TOON compression.
---

## Overview

This endpoint proxies to the OpenAI Chat Completions API (or any OpenAI-compatible upstream set via `UPSTREAM_URL`). toongate compresses uniform arrays in `messages[].content` before forwarding the request.

The response is identical to what OpenAI returns — including streaming SSE when `stream: true`.

## Authorization

<ParamField header="Authorization" type="string" required>
  `Bearer <key>`. toongate injects `OPENAI_API_KEY` automatically — you may pass your key here or rely on the injected value.
</ParamField>

## Request body

The request body is the standard [OpenAI Chat Completions request](https://platform.openai.com/docs/api-reference/chat/create). All fields pass through unchanged.

<ParamField body="model" type="string" required>
  Model ID (e.g. `gpt-4o`, `gpt-4o-mini`).
</ParamField>

<ParamField body="messages" type="array" required>
  Array of message objects. `content` fields containing uniform arrays of objects are eligible for TOON compression.
</ParamField>

<ParamField body="stream" type="boolean">
  When `true`, returns a stream of SSE events. toongate compresses the request payload and passes chunks back as-is.
</ParamField>

## Response headers

| Header | Description |
|---|---|
| `X-Toongate-Compressed` | `true` if the request payload was compressed |
| `X-Toongate-Tokens-Before` | Estimated token count before compression |
| `X-Toongate-Tokens-After` | Estimated token count after compression |
| `X-Toongate-Tokens-Saved` | Tokens saved |
| `X-Toongate-Eligibility-Score` | Eligibility score (0–1) |

<RequestExample>
```bash cURL
curl https://toongate.workers.dev/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-..." \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {
        "role": "user",
        "content": "Summarize: [{\"id\":1,\"title\":\"A\",\"score\":0.9},{\"id\":2,\"title\":\"B\",\"score\":0.8}]"
      }
    ]
  }'
```
</RequestExample>
