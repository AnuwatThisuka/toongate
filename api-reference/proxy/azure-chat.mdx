---
title: "POST /azure/v1/chat/completions"
api: "POST https://toongate.workers.dev/azure/v1/chat/completions"
description: Azure OpenAI chat completions with automatic TOON compression.
---

## Overview

Proxies to `{AZURE_OPENAI_ENDPOINT}/openai/deployments/{model}/chat/completions?api-version={AZURE_OPENAI_API_VERSION}`. The deployment name is taken from the `model` field. toongate injects the `api-key` header automatically.

## Authorization

<ParamField header="Authorization" type="string">
  Required only when `PROXY_AUTH_KEY` is set on the Worker. Pass `Bearer <PROXY_AUTH_KEY>` to authenticate to toongate itself. The Azure API key (`AZURE_OPENAI_API_KEY`) is always injected automatically as the `api-key` header — you do not need to pass it.
</ParamField>

## Request body

The request body is the standard OpenAI Chat Completions request schema.

<ParamField body="model" type="string" required>
  Azure **deployment name** (e.g. `my-gpt4o-deployment`). toongate uses this to build the upstream URL: `.../deployments/{model}/chat/completions`.
</ParamField>

<ParamField body="messages" type="array" required>
  Array of message objects. `content` must be an array (not a plain string) for structured data to be eligible — use OpenAI multimodal format with `{ type: "text", text: "..." }` parts.
</ParamField>

<ParamField body="stream" type="boolean">
  When `true`, returns SSE events. toongate compresses the request and passes chunks back as-is.
</ParamField>

## Response headers

| Header | Description |
|---|---|
| `X-Toongate-Compressed` | `true` if the request payload was compressed |
| `X-Toongate-Tokens-Before` | Estimated token count before compression |
| `X-Toongate-Tokens-After` | Estimated token count after compression |
| `X-Toongate-Tokens-Saved` | Tokens saved |
| `X-Toongate-Eligibility-Score` | Eligibility score (0–1) |

<RequestExample>
```bash cURL
curl https://toongate.workers.dev/azure/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $PROXY_AUTH_KEY" \
  -d '{
    "model": "my-gpt4o-deployment",
    "messages": [
      {
        "role": "user",
        "content": [
          { "type": "text", "text": "Summarize:" },
          { "type": "text", "text": "[{\"id\":1,\"title\":\"A\",\"score\":0.9},{\"id\":2,\"title\":\"B\",\"score\":0.8}]" }
        ]
      }
    ]
  }'
```
</RequestExample>
