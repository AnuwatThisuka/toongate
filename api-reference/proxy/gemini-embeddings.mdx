---
title: "POST /gemini/v1/embeddings"
api: "POST https://toongate.workers.dev/gemini/v1/embeddings"
description: Google Gemini embeddings via OpenAI-compatible endpoint with TOON compression.
---

## Overview

Proxies to Gemini's OpenAI-compatible embeddings endpoint (`https://generativelanguage.googleapis.com/v1beta/openai/embeddings`) or a custom `GEMINI_UPSTREAM_URL`.

## Authorization

<ParamField header="Authorization" type="string">
  Required only when `PROXY_AUTH_KEY` is set. Pass `Bearer <PROXY_AUTH_KEY>`. `GEMINI_API_KEY` is injected automatically.
</ParamField>

## Request body

<ParamField body="model" type="string" required>
  Gemini embedding model ID (e.g. `text-embedding-004`).
</ParamField>

<ParamField body="input" type="string | array" required>
  Text or array of texts to embed. Arrays of structured objects are eligible for TOON compression.
</ParamField>

## Response headers

| Header | Description |
|---|---|
| `X-Toongate-Compressed` | `true` if the request payload was compressed |
| `X-Toongate-Tokens-Before` | Estimated token count before compression |
| `X-Toongate-Tokens-After` | Estimated token count after compression |
| `X-Toongate-Tokens-Saved` | Tokens saved |

<RequestExample>
```bash cURL
curl https://toongate.workers.dev/gemini/v1/embeddings \
  -H "Content-Type: application/json" \
  -d '{
    "model": "text-embedding-004",
    "input": "The quick brown fox"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "object": "list",
  "data": [{ "object": "embedding", "embedding": [0.01, -0.02, ...], "index": 0 }],
  "model": "text-embedding-004",
  "usage": { "prompt_tokens": 5, "total_tokens": 5 }
}
```

```json 503
{ "error": "Gemini is not configured" }
```
</ResponseExample>
